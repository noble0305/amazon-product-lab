# 会话交接

## 已确认的创业方向

- 目标是一人公司，获得收入自主权，逐步替代工资。
- 12 个月目标为月稳定利润 2 万至 3 万元人民币。
- 已有亚马逊美国站和英国站店铺，但缺乏系统化选品经验。
- 第一阶段只做美国站，以自有店铺和真实资金验证。
- 验证成功后再考虑将系统产品化，不融资、不扩团队优先。

## 产品定位

建立“AI 亚马逊选品实验室”：以商机探测器为需求入口，以真实销售结果为反馈标签，形成机会发现、打样、上架、销售和复盘闭环。

首期预算 2 万至 5 万元。优先测试售价 20 至 50 美元、轻小件、非季节性、低认证门槛商品。每款小批量采购，人工拥有最终决定权。

“高胜率”定义为经过足够真实实验后，Agent 推荐持续优于历史人工基准，不代表保证爆款。

## 当前计算口径

- 广告每单成本：`CPC / 转化率`
- 基准贡献利润包含：售价、到岸成本、FBA、佣金、仓储、预计退货损失和广告成本
- 基准贡献利润率低于 15% 时直接拒绝
- 总分 80 以上进入打样，65 至 79 进入观察池
- 高合规风险、高侵权风险或危险品直接拒绝

这些是第一版实验假设，不应当被当成行业真理。必须通过真实商品回测后调整。

## 下一步优先级

1. 从 ASIN 产品机会榜中选择 10 至 20 个参考商品，形成 3 至 5 个产品方案。
2. 为产品方案补录供应商报价、尺寸重量、FBA、CPC、转化率和风险数据。
3. 将产品方案接入现有三情景利润模型和 15% 利润红线。
4. 增加人工通过、拒绝、拒绝理由和询价/打样状态。
5. 用 3 个历史商品盲测利润与评分结果。
6. 之后再实现评论和退货原因聚类，不提前开发自动上架。
